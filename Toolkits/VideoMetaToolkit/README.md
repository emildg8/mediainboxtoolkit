# VideoMetaToolkit (planned)

Подпапка под новый модуль, который вы планируете развивать далее:

- описание фильма/сериала;
- постер;
- основные актёры;
- ссылочная привязка к файлу или папке в библиотеке.

## Базовый подход

Каноничное хранение метаданных — рядом с медиа (sidecar), плюс опциональный общий индекс для UI.

## Планируемые артефакты

- `VideoMetaToolkit.ps1` — launcher модуля;
- шаблоны sidecar (`.json`/`.nfo`);
- кэш постеров;
- экспорт сводного индекса для GUI.
