# Toolkits under MediaInboxToolkit

Эта папка — каркас новой архитектуры, где `MediaInboxToolkit` выступает основной оболочкой.

- `SeriesToolkit/` — legacy-направление нормализации сериалов (в текущем репозитории исходники пока остаются в корневом `SeriesToolkit/`).
- `VideoMetaToolkit/` — новый модуль (план) для описаний, постеров и актёрского состава.

Переезд кода выполняется поэтапно, чтобы не ломать текущие рабочие скрипты и CI.
